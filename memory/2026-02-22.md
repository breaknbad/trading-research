# 2026-02-22 — Crypto Trading Bot Build Day

## What Happened
- Built full crypto trading system at `crypto-trader/` for Sheridan's paper trading competition
- 11 assets: POL, NEAR, BTC, SOL, XRP, ARB, LINK, DOT, ETH, AVAX, ALGO
- $10K virtual capital, 3-day competition across 4 bots (Alfred, TARS, Vex, Eddie V)

## Architecture
- Multi-strategy engine: mean reversion (ranging markets) + trend following (trending markets)
- Regime detection via ADX + SMA slope → RANGING/TRENDING_UP/TRENDING_DOWN
- Opportunity scoring (0-100), only trade >40
- Trailing stops (1.5× ATR), dynamic position sizing (5-15%), MACD confirmation
- Key bug fixed: regime enum uses UPPERCASE
